## GitHub issue templates

### 1. 라벨
	- front: 프론트엔드 기능 개발
	- back: 백엔드 기능 개발
	- fix: 에러 발견 및 수정
	- refactor: 리팩토링
	- chore: 요구사항 변경
	- gui: UI 개발
	- build: 빌드 및 외부 라이브러리
