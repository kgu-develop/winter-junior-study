## Markdown

### 정의

- 일반 텍스트 문서에 서식 요소를 추가하는 데 사용할 수 있는 가벼운 마크업 언어

### 장점

1. 간결성
2. 별도의 도구없이 작성가능
3. 다양한 형태로 변환이 가능
4. 텍스트로 저장 → 가벼움 + 버전 관리 시스템을 이용 가능 (변경 이력 관리 가능)

### 단점

1. 표준의 부재 → 도구에 따라 변환 방식이나 생성물에 차이
2. 모든 HTML 마크업 대체 불가능

### 문법

- Heading
    
    # H1
    ## H2
    ### H3
    
- Bold
    
    **bold**
    
- Italic
    
    *italic*
    
- BockQuote
    
    > blockquote
    
- Ordered List
    
    
    1. First item
    2. Second item
    3. Third item
    
    
- Unorderd List
    
    
    - First item
    - Second item
    - Third item
    
    
- Code
    
    ``` code ``` 
    
- Horizonatal
    
    ---
    
- Link
    
    [title](https://www.example.com)
    
- Image
    
    ![alt text] (image.jpg)
